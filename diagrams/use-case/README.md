# Use Case Diagrams

This folder contains use case diagrams for InfinityTalk application.

## Diagrams

- `infinitytalk-use-cases.md` - Main use cases diagram (Mermaid format)
- `infinitytalk-use-cases.puml` - Main use cases diagram (PlantUML format) - **Use this for planttext.com**

## How to use PlantUML version

1. Go to https://www.planttext.com/
2. Copy the content of `infinitytalk-use-cases.puml`
3. Paste it in the PlantText editor
4. The diagram will be rendered automatically

## Actors

- **User**: Regular user with basic permissions
- **Manager**: User with project management capabilities
- **Administrator**: User with full system administration rights

## Use Case Packages

- **Authentication**: User registration and login
- **Messaging**: Communication features
- **Calls & Meetings**: Audio/video communication
- **File Management**: File upload/download/sharing
- **Project Management**: Project and task management
- **Administration**: System administration features
